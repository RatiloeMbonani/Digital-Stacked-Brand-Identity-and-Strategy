# Digital Stacked – Brand Identity & Strategy

I created **Digital Stacked**, a personal branding project designed to position myself as a **Full Stack Developer in the mining industry**. The project focuses on building a strong visual identity, brand strategy, and supporting documentation to reflect technical and professional growth.

## Project Overview

This project includes the development of a complete brand system, from research and strategy to visual identity and documentation.

## What’s Included

- Brand Strategy  
- Brand Guide  
- Logo Design  
- Research Report  
- Documentation 


## Tools Used

- Adobe Illustrator  
- Adobe InDesign  

## Purpose

The goal of this project is to establish a professional identity as a full stack developer within the mining industry, combining creativity, technical skills, and strategic thinking.
